# HANDMIXED

HANDMIXED is a browser-based DJ mixing platform that explores alternative ways of interacting with digital music through computer vision and gesture recognition. Rather than relying on traditional DJ hardware, the system allows users to control mixing functions using hand movements captured by a webcam.

The project was built using MediaPipe hand tracking alongside web technologies to create an interactive dual-deck mixing experience. HANDMIXED focuses on combining human-computer interaction principles with music technology, aiming to create a more intuitive and accessible approach to digital audio control.

Current functionality includes track loading, gesture-based interaction, and real-time hand tracking integration. The project also explores how computer vision can be applied to creative software environments beyond conventional input methods.

## Technologies Used

- JavaScript
- HTML / CSS
- MediaPipe
- Spotify API Integration
- Browser Audio Processing

HANDMIXED was developed as an exploration of computer vision, interactive system design, and digital audio technology, with the goal of creating a hands-free music mixing experience.
